프로필

안녕하세요, 저는 김혜림입니다. 데이터베이스 및 서버 구축에 대한 전문성을 가지고 있으며, 프론트엔드 개발에 대한 경험도 있습니다.

다양한 기술 스택을 보유하고 있으며, 문제 해결 능력과 협업 능력이 뛰어난 개발자입니다.

경력

ABC 회사 - 백엔드 개발자

기간: 2020.01 ~ 2023.12

주요 업무: RESTful API 개발, 데이터베이스 설계 및 최적화, 클라우드 기반 서비스 구축

XYZ 회사 - 프론트엔드 개발자

기간: 2018.06 ~ 2019.12

주요 업무: 웹 애플리케이션 UI/UX 디자인, JavaScript 및 React를 이용한 프론트엔드 개발

프로젝트

온라인 쇼핑몰 개발

사용된 기술: Spring Boot, MySQL, Thymeleaf

프로젝트 설명: 사용자가 상품을 검색하고 장바구니에 담아 결제할 수 있는 기능을 구현했습니다.

깃허브 링크

개인 블로그 웹사이트

사용된 기술: Node.js, Express, MongoDB

프로젝트 설명: 사용자가 블로그 글을 작성하고 댓글을 남길 수 있는 웹 애플리케이션을 개발했습니다.

깃허브 링크

기술

  • Java (Spring Boot)
  • JavaScript (React, Node.js)
  • Python (Flask, Django)
  • MySQL, MongoDB
  • HTML5, CSS3
  • RESTful API 개발

학력

서울대학교 - 컴퓨터공학과

졸업년도: 2018

학습한 주요 과목: 알고리즘, 데이터베이스, 웹 개발, 시스템 프로그래밍
